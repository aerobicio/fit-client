# Fit::Client

A ruby gem for interacting with the aerobic.io fit-service.

## Installation

Add this line to your application's Gemfile:

    gem 'fit-client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fit-client

## Usage

Don't use this. The fit-service is private.
